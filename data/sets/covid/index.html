COVID Calculator

If COVID-19 continues to grow exponentially in the US, we will exhaust hospital beds within 3 weeks:

NewBedsNeeded = \(CurrentCases\) \(\cdot\) \(DailyGrowthRate\) ^ \(DaysOfGrowth\) \(\cdot\) \(SevereAndCriticalRate\)
Reference:

\(NewBedsNeeded\) = Number of new hospital beds needed for Severe and Critical Cases.

\(CurrentCases\) = The number of COVID-19 cases in the US as of March 9, 2020.

\(DailyGrowthRate\) = Number of today's cases / Number of yesterday's cases.

\(SevereAndCriticalRate\) = Percentage of cases requiring a hospital bed.

\(DaysOfGrowth\) = How many days to run the simulation.

Reference = Source of the parameters.
